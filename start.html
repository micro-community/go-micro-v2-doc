--- title: "Start using M3O in minutes" description: "M3O getting started guide" layout: default ---

Install Micro to build, run and manage services locally or on the M3O platform. 

go get github.com/micro/micro/v3

This command makes the rest of your CLI commands run on the M3O platform. 

micro env set platform

Signup is required before you can use M3O. 

micro signup

Login with the credentials provided during signup. 

micro login

Create a new service (follow the instructions and push to a public repo on Github) 

micro new helloworld

Deploy the service (we're using an example helloworld service) 

micro run github.com/micro/services/helloworld

See if your services are running. 

micro status

See its logs. 

micro logs helloworld

Call it. 

micro call helloworld Helloworld.Call '{"name":"Alice"}'

Deploy new changes 

micro update github.com/micro/services/helloworld

For more info see m3o.dev or join #m3o-support on slack for help.
