About

 

Micro is a cloud native platform built for developers

Overview

The world is moving towards becoming entirely driven by technology. Building software for billions of people is hard. With the emergence of Cloud computing we've accelerated software delivery but software development tools have largely lagged behind. At Micro we've developed a model for effectively leveraging the Cloud and now want to share that with the world.

Design

Micro is a fully managed cloud platform built for developers. We provide a strongly opinionated development model based on the microservices software design pattern and abstract away the complexity of the cloud so developers can focus on building products rather than managing infrastructure.

Architecture

A Micro services architecture enables products to be broken down into smaller services which teams work on and scale independently of each other. Each team becomes responsible for the delivery of their particular service which others come to use through common APIs. In this model everything is a service.

Technology

We want to empower developers to build software at an accelerated pace from early stage prototyping all the way through to large scale production deployment without having to think about managing the infrastructure. Micro aspires to become host to the next generation of services through a cloud native development platform.
